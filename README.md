# bbtbar (Forge 1.20.1)
Client-side HUD overlay mod that replaces vanilla hearts with an 80x9 health bar.

## Where to draw your textures
Replace PNGs here:
`src/main/resources/assets/bbtbar/textures/gui/healthbars/`

Files:
- full.png
- wither.png
- poison.png
- frozen.png
- intermediate.png
- empty.png
- absorption.png

All of them are **80x9**. Keep the same size + file names.

## Run
- Java 17
- Forge 1.20.1 (47.2.20)

In terminal:
- `gradle runClient`  (or install Gradle, or add a full wrapper via `gradle wrapper`)
